# GitHub Repositories Explorer

🚀 A **React + TypeScript** app that allows users to explore GitHub repositories using the GitHub API.

## 🛠️ Technologies Used
- ⚛️ React + TypeScript  
- 🐙 GitHub API  
- 🎨 Tailwind CSS  
- 🔧 Lucide React Icons  

## 📌 Features
- 🔍 Search GitHub repositories by **username**  
- ⭐ View repository **star count**
